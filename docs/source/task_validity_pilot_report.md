# Task Validity Pilot Report

## Purpose

Record the first governed evidence on whether MarketMind tasks are leakage-safe and non-exchangeable enough to justify trainer commitment.

## Questions This Report Must Answer

- Are support/query boundaries leakage-safe under the planned task geometry?
- Do regime-indexed tasks behave as non-exchangeable learning units?
- Does the null distribution from shuffled labels or shuffled regime assignments collapse as expected?
- Are any hidden assumptions still preventing honest trainer commitment?

## Minimum Evidence

- leakage-geometry diagnostics,
- non-exchangeability pilot results,
- baseline comparison notes relevant to W2,
- provisional thresholds still marked `⚑ VALIDATE`,
- explicit recommendation: proceed, narrow, or stop.

## Phase Relationship

- Phase I-G owns the pilot design and proof burden.
- Phase II-0 owns the reproducible harness and artifact emission.
- Phase II trainer commitment remains gated on this evidence being clean enough.
